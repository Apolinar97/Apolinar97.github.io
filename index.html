An Android application that allows users to create music channels. The channel owner is able to recieve music requests and can either add them to the queue or deny them. Requesters can request music to be played in the music channel.

Trending Stocks

A bot that gathers trending stocks from StockTwits and gets their metadata from Finnhub API.

The bot consolidates the trending stocks listing its Symbol, Open Price, Current Price, and Previous Close Price into a table.

A simple linear regression model trained using 2020 contemporary covid-19 data.

Diabetes Logistic Regression Model with console interface

Trained a logistic regression model to be able to predict if a person has diabetes.

The model is hosted on a webserver making it accessible to users. The model can be queried with data inputs such as glucose level, BMI levels,etc. to predict if an individual has diabetes.
